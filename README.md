### 💰 FinanceTracker — Offline-First Personal Finance Workspace

A modern personal finance platform designed with an **Offline-First Architecture**, enabling users to manage budgets, expenses, and financial goals seamlessly—even without internet connectivity.

#### ✨ Key Features

* 📱 Offline-first experience using Service Workers, IndexedDB, and Workbox
* 💵 Expense tracking and budget management with local-first processing
* 🤝 Collaborative budgeting workspaces for couples, roommates, and teams
* 📊 Dual budgeting system:

  * Monthly Planned Workspace
  * Auto-generated "From Plans" Workspace
* 🔥 Gamified savings challenges, streaks, and achievement badges
* 🔐 OTP-based authentication and secure data management
* ☁️ Background synchronization with cloud backup services
